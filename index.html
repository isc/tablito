Tablito

Apprendre les tables de multiplication, en douceur.

  • 100 % gratuit
  • dès le CE1 · 7–11 ans
  • tables & division
  • Méthode Leitner. Chaque multiplication revient juste avant que l'enfant l'oublie — la mémoire s'ancre durablement.
  • Pas de notes, pas de classement. On célèbre le progrès, pas la performance. Aucun message négatif.
  • Hors-ligne, sans pub, sans compte. Une fois installé, plus besoin de connexion. Les données restent sur l'appareil.

L'app s'installera comme n'importe quelle autre application.

Attention : si tu installes Tablito plus tard sur iPhone ou iPad, la progression du navigateur ne sera pas conservée.

Installer sur iPhone / iPad

Ouvre cette page dans Safari, puis ajoute l'app à l'écran d'accueil :

  1. Touche le bouton ••• en bas à droite de Safari (en mode compact, c'est lui qui cache le partage).
  2. Choisis « Partager » dans le menu qui s'ouvre.
  3. Touche « Plus » pour afficher toutes les actions.
  4. Choisis « Sur l'écran d'accueil » puis touche « Ajouter ». L'icône Tablito apparaît sur ton écran.
  1. Touche le bouton de partage en bas de l'écran (icône avec une flèche vers le haut).
  2. Choisis « Sur l'écran d'accueil » dans la liste.
  3. Touche « Ajouter » en haut à droite. L'icône Tablito apparaît sur ton écran d'accueil.

Astuce : dans Chrome ou Firefox sur iOS, l'ajout n'est pas disponible — ouvre cette page dans Safari.
